Answer:
dma1: Direct Memory Access controler. Allows peripheral device work with
main memory without accessing CPU
pic1: Procrammable Interupt controler. Allows manage the interupts
for execution(shedule them)(https://en.wikipedia.org/wiki/Programmable_interrupt_controller) 
timer1: Now, this is hardware interupt timer. Allows, for example, trigger
interrupt when pressing a keyboard or moving a mouse (Wikipedia:
https://en.wikipedia.org/wiki/Interrupt#Hardware_interrupts)
